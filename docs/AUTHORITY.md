# Authority

## Scope

This repository defines the supreme law of ALIVE. All other repositories are bound by this authority.

## Allowed Actions

- Defining invariants and bounds
- Creating and updating contracts
- Establishing identity rules
- Writing policy definitions
- Processing amendments
- Exporting law for other repositories

## Forbidden Actions

- Executing code or performing runtime operations
- Performing cognition or reasoning
- Storing runtime state
- Direct external interaction
- Modifying without formal amendment

## Enforcement Points

- Runtime enforces constitutional rules
- Body enforces boundary constraints
- Mind operates within constitutional bounds

## Cross-Repo Interaction Rules

- Other repos may import from constitution
- Constitution imports nothing
- All changes to constitution require amendment
- No runtime override permitted

## Escalation Rules

- Issues escalate to human authorization
- Constitutional violations trigger system alerts
- Emergency conditions do not override constitution
